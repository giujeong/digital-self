# Digital Self — 정기우

> 막연함에서 명료함으로

## About

40을 앞둔 지금, 나를 정의하는 디지털 셀프 프로젝트.

- **믿는 자** — 넘어지되 다시 일어나는 자
- **질문하는 자** — 문제 푸는 사람
- **확신** — 참 인간은 주님의 임재 안에서 가능하다

## Tech Stack

- React + Vite
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
